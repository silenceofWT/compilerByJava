### 文件说明：
- UI文件夹 为界面设计文件夹 暂未实现
- icon 为界面设计使用到的图标文件
- designFile文件夹 为文法设计和LL1分析表
- inputFile文件夹 存储源代码的输入文件
1. inputCode.txt 为要运行的源代码。
- outputFile文件夹 存储运行过程中输出的结果
1. Tokens.txt 会输出Token序列
2. quaternaryExp.txt 会输出四元式序列
CommonUtils.java 存放所有类用到的公共组件或工具
Main.java 运行的主函数

block.java 基本块划分类

fileParseUtils.java 文件解析工具类

lexicalAnalyzer.java 词法分析类

parser.java 语法分析类

quaternaryExpression.java 四元式生成类

table.java 符号表类
